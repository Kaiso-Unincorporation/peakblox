# Building Android APK for PeakBlox

## Options:
1. **Kivy / Buildozer** (Python-based):
   - Install Buildozer
   - Create main.py with similar UI
   - `buildozer -v android debug`

2. **React Native / Flutter** wrapper around web version.

3. **WebView APK**: Simple wrapper loading the PeakBlox site.

For quick start, use WebView template or sideload the web app as PWA.

Full APK build requires Android SDK. See Buildozer docs.
