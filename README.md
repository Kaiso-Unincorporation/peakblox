# PeakBlox - Classic Roblox Revival Platform

Inspired by caelus.lol. Personal/private server for old Roblox experiences.

## Features
- Web-based PeakBlox Studio (mock Roblox Studio)
- Custom Launcher (EXE for Windows)
- APK for Android
- Discord auth ready
- Classic games support via Novetus/Rboxlo2
